# -Future-Equity-System-
一个创新的、公平的公司股权制度设计，将人力资本置于金融资本之上。(An innovative and equitable corporate equity system design that prioritizes human capital over financial capital)

明文规定下；除去工资和每年按比例预留的一份资金（用于抵抗危机，购买在外的A股与承担已收回的A股的后续设备费；资金会堆叠且有按比例的上限，多出的分到B股）外的利润分为AB股；初始A股占利润的一半且无实权，用于售卖（换取启动资金），但需持有者同时按比例提供给公司设备的购买与维护费（可选择不提供，但会影响自身利益，可协商）；初始B股占利润的一半且有实权，员工获得重大成果时会获得一份（全程公开可提出质疑）（其余股份会留在公司），由创始人与后续的审查组审核与发放，且仅有股份较高的能参与重要投票；除了买回的A股会变为B股外，AB股持有者死后的股份也会被公司按约定价强制回购为B股用于分配；有且仅有创始人有一票否决权直到死亡（仅用于应对初其关键决策）
